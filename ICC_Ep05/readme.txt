Nesta atividade foram implementados dois métodos diferentes para o cálculo da área da função Styblinski-Tang: O método dos retângulos e o método de Monte Carlo. 
Algumas comparações foram feitas:
- Comparando-se os resultados para n = 2 dimensões nos dois métodos, observa-se que quanto mais amostras são utilizadas, mais o resultado se aproxima do resultado real.
    Foi observado que com 10^8 amostras, ambos os métodos retornam um valor satisfatoriamente aproximado.
- Comparando-se os resultados para n = 2, 4 e 8 dimensões para o método de monte carlo, quanto maior o número de dimensões, maior o resultado fica.
